# v1.3.3-Q1-selected-static-candidate-status

## Scope

v1.3.2 proved that at least one screened Q1_Zn motif can remain valid after LAMMPS static minimization.

v1.3.3 promotes the robust static candidate-selection logic into the default Q1_Zn generation path. The selection is deterministic and based on pre-minimization topology and local motif geometry. It does not hard-code a single atom ID and does not require running the screening workflow during normal generation.

## Default Q1 Selection

Default mode:

```text
PYCSH_ZN_Q1_SELECTION_MODE=ranked_static
```

`ranked_static` enumerates topology-valid Q1 candidates and ranks them by:

- safe terminal/non-bridging hydroxylation preconditions;
- ZnO2(OH)2-like motif geometry score;
- max and mean intended Zn-O distances;
- O-Zn-O tetrahedral angle deviation from 109.47 degrees;
- minimum O-O distance among intended motif O atoms;
- secondary-shell static oxygen crowding diagnostics;
- deterministic seed-based tie breaker.

The normal generator does not use post-min results. `examples/10_screen_q1_motifs.py` remains the explicit screening/diagnostic workflow.

The selected atom id is a result of deterministic static ranking for the current seed/base structure, not a hard-coded requirement.

## Mechanics Policy

Q1 post-min acceptance is still required before Q1 mechanics. Q1 mechanics remains disabled by default in `examples/07_run_quasistatic_mechanics.py`.

## Boundaries

- CementFF4-Zn force-field parameters are unchanged.
- The 2.5 A Zn-O coordination validation gate is unchanged.
- Validation semantics are unchanged.
- No finite-temperature MD is introduced.
- No MD-ready classification semantics are introduced.
- mixed Q1+Q2b is not introduced.
- Q1 is not added to default quasi-static mechanics.
