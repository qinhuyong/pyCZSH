# pyCSH-Zn v1.3-Q1-Zn-static Status

Base release:

```text
v1.2-quasistatic-mechanics
```

Scope:

- Conservative static Q1_Zn candidate generation path.
- Pure C-S-H, Q2b_Zn, and Q1_Zn static candidates.
- No finite-temperature MD.
- No core-shell MD protocol development.

Stage A audit:

- Existing Q1-like sites were identified by pyCSH brick piece labels `<L`, `<R`, `>L`, `>R`, `<Lo`, `<Ro`, `>Lo`, and `>Ro`.
- Existing Q2b-like sites were identified by `SU`, `SD`, `SUo`, and `SDo`.
- The old Q1 path shared the Q2b substitution/hydroxylation machinery and could generate a candidate, but it was prototype-level because it lacked Q1-specific candidate reporting and validator classification.
- Reused: safe terminal/non-bridging O screening, H placement, O/S shell-to-Oh/Hoh conversion, Zn angle remapping, and CementFF4-Zn output.
- Rewritten/exposed: candidate precondition report, Q1 classification, Q1 examples, Q1 static relaxation runner, opt-in mechanics guard, and documentation.

Motif assumption:

- Q1_Zn uses the existing strict static substitutional idea: replace a Q1/terminal silicate Si center with Zn(+2), then hydroxylate two safe terminal/non-bridging O core-shell pairs into Oh-Hoh using CementFF4-Zn angle remapping.
- This is a conservative static candidate, not a unique experimental Q(1,Zn) assignment.

Validation:

- Q1_Zn generates as `valid_q1_zn_candidate`.
- After static minimization, the current conservative motif may relax only to `needs_static_relaxation`; this is a diagnostic outcome, not a final success claim.
- MD-ready classification is not used.
- Finite-temperature MD is not run.

Acceptance results:

- `examples/01_generate_pure_csh.py`: passed.
- `examples/02_generate_q2b_zn.py`: passed.
- `examples/08_generate_q1_zn.py`: passed, final classification `valid_q1_zn_candidate`.
- `examples/03_validate_outputs.py`: passed for pure C-S-H, Q2b_Zn, and Q1_Zn.
- `examples/04_build_lammps_inputs.py`: passed and generated Q1_Zn LAMMPS input templates when Q1 data was present.
- `examples/06_run_static_relaxation.py`: passed for the existing v1.1 pure C-S-H and Q2b_Zn targets.

Limitations:
