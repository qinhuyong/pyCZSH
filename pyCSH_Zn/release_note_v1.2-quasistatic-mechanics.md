# pyCSH-Zn v1.2-quasistatic-mechanics Release Note

v1.2 provides a controlled quasi-static mechanics pipeline validation for pure
C-S-H and one Q2b_Zn static candidate. It starts from the v1.1 static-relaxed
structures, runs x-direction small-strain deformation/minimization cases, and
writes mechanics CSV/JSON summaries plus SVG plots using actual strain.

This release is not a final elastic-constant calculation, not a production
mechanical-property workflow, and not finite-temperature MD.
